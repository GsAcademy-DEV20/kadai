## そのほかメモ
- PHPのバージョン確認  
```php -v```
- composerのバージョン確認  
```composer -v```
インストールされてなかったら```composer install```

- Laravelのインストール  
```$ composer create-project laravel/laravel TodoApp```  
TodoAppはアプリ名

- ローカルサーバの起動  
```php artisan serv```

- laravel-ide-helperのインストール  
```$ composer require --dev barryvdh/laravel-ide-helper```
